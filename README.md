# 🧮 Modern Calculator

A simple **Modern Calculator Web Application** built using **HTML, CSS, and JavaScript**.
This calculator allows users to perform basic arithmetic operations such as **addition, subtraction, multiplication, and division** with a clean and responsive interface.

---

## 📌 Features

* Perform **basic arithmetic operations**

  * Addition (+)
  * Subtraction (-)
  * Multiplication (*)
  * Division (/)
* Modern **gradient background UI**
* Styled **calculator card layout**
* Handles **division by zero**
* Interactive **hover button animation**
* Displays result instantly on the webpage

---

## 🛠 Technologies Used

* **HTML5**
* **CSS3**

## ⚙️ How It Works

1. The user enters **two numbers**.
2. The user selects a **mathematical operator**.
3. When the **Calculate** button is clicked:

   * JavaScript reads the input values.
   * A **switch statement** performs the selected operation.
4. The result is displayed in the **Result box**.

Example:

```text id="3xt3ad"
First number: 10
Second number: 5
Operator: +

Result: 15
```
